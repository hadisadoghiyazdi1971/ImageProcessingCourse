# فصل سوم کتاب «پردازش تصویر در دهه آینده»

## عنوان فصل

**سامانه‌های خطی، تحلیل فوریه و نمایش چندمقیاسی**

این بسته یک فصل مستقل و پژوهش‌محور برای درس پردازش تصویر در مقطع کارشناسی ارشد است. متن از تعریف عملگر خطی و انتقال‌ناوردا آغاز می‌شود و تا موجک، فضای مقیاس، چارچوب‌های جهت‌دار، پراکندگی موجکی، ضدعلیاسینگ شبکه‌های عمیق، سوگیری طیفی، Fourier Features، SIREN و Fourier Neural Operator ادامه می‌یابد.

## فایل‌ها

- `image_processing_next_decade_ch03_fa.tex`: نسخه مستقل فصل، دارای صفحه عنوان، فهرست مطالب، منابع و تنظیمات XePersian.
- `chapter03_lti_fourier_multiscale_content_fa.tex`: محتوای فصل برای `\input` در فایل مادر کتاب.
- `image_processing_next_decade_ch03_fa.pdf`: خروجی نهایی ۵۳ صفحه‌ای.
- `generate_ch03_figures.py`: بازتولید شکل‌ها و آزمایش‌های عددی.
- `experiment_results.csv`: خروجی خام عددی آزمایش‌ها.
- `experiment_results.tex`: ماکروهای عددی مورد استفاده در متن؛ اعداد جدول مستقیماً از این فایل فراخوانی می‌شوند.
- `source_audit_ch03.md`: ممیزی منابع، استانداردها و مقالات اصلی.
- `requirements.txt`: وابستگی‌های پایتون.
- `figures/`: شکل‌های PDF و PNG تولیدشده با کد.

## کامپایل متن

از پوشه فصل دو بار XeLaTeX را اجرا کنید:

```bash
xelatex -interaction=nonstopmode image_processing_next_decade_ch03_fa.tex
xelatex -interaction=nonstopmode image_processing_next_decade_ch03_fa.tex
```

در TeXstudio، موتور پیش‌فرض را روی **XeLaTeX** قرار دهید. فونت اصلی متن `Amiri` است.

## اجرای آزمایش‌ها

```bash
python -m pip install -r requirements.txt
python generate_ch03_figures.py
```

اسکریپت شکل‌ها را در `figures/` و نتایج را در `experiment_results.csv` و `experiment_results.tex` بازنویسی می‌کند. پس از اجرای آن، فایل LaTeX را دوباره کامپایل کنید.

## ویژگی بازتولیدپذیری

- برای داده‌ها و شبکه کوچک آزمایش سوگیری طیفی seed ثابت استفاده شده است.
- جدول نتایج به‌صورت دستی تایپ نشده و از ماکروهای تولیدشده در `experiment_results.tex` تغذیه می‌شود.
- شرایط مرزی کانولوشن مستقیم و FFT یکسان‌سازی شده‌اند؛ مقایسه مرز متقارن با مرز صفر فقط در ناحیه داخلی انجام می‌شود.
- آزمایش MTF لبه مایل، **نمایش آموزشیِ ساده‌شده** است و ادعای انطباق آزمایشگاهی کامل با ISO 12233:2024 ندارد.
- تصاویر واقعی آزمایشی از داده‌های نمونه همراه `scikit-image` گرفته شده‌اند. برای انتشار تجاری یا بازنشر جداگانه شکل‌ها، مجوز تصویر نمونه و سیاست مجله بررسی شود.
