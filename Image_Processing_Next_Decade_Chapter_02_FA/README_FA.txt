فصل دوم کتاب «پردازش تصویر در دهه آینده»
================================================

عنوان فصل: دیجیتالی‌شدن جهان دیداری
نسخه: تفصیلی، ۵۱ صفحه، تابستان ۱۴۰۵ / ۲۰۲۶

فایل‌های اصلی
-------------
1) image_processing_next_decade_ch02_fa.tex
   نسخه مستقل کامل با صفحه عنوان، فهرست مطالب، فصل و منابع.

2) image_processing_next_decade_ch02_fa.pdf
   خروجی نهایی که با XeLaTeX کامپایل و صفحه‌به‌صفحه رندر و بررسی شده است.

3) chapter02_digitization_content_fa.tex
   فقط محتوای فصل دوم، برای قرار دادن در فایل مادر کتاب با دستور input.

4) chapter02_references_fa.tex
   منابع فصل دوم به‌صورت محیط thebibliography.

5) generate_ch02_figures.py و پوشه figures
   کد بازتولید شش شکل اصلی فصل و تصاویر تولیدشده.

6) SOURCE_AUDIT_CH02.md
   فهرست منابع اصلی وب، استانداردها و دلیل استفاده از هر منبع.

روش کامپایل
-----------
- TeXstudio را باز کنید و Compiler را روی XeLaTeX قرار دهید.
- فایل image_processing_next_decade_ch02_fa.tex را دو بار کامپایل کنید.
- پوشه figures باید کنار فایل tex باقی بماند.
- فونت‌های استفاده‌شده: Amiri و DejaVu Serif/Mono.

بازتولید شکل‌ها
---------------
در محیط پایتون:
    pip install numpy scipy matplotlib pillow scikit-image
    python generate_ch02_figures.py

نکات علمی
---------
- روابطی مانند Delta^2/12 و 6.02b+1.76 dB همراه با فرض‌های اعتبارشان آمده‌اند.
- داده خطی از داده sRGB/PQ/HLG جدا شده است.
- نویز RAW به‌صورت پواسون ـ گاوسی و دامنه دینامیکی بر پایه مدل حسگر نوشته شده است.
- آخرین نسخه استاندارد HDR استفاده‌شده BT.2100-3، منتشرشده در فوریه ۲۰۲۵ است.
- مقاله CVPR 2025 درباره Alias-Free Latent Diffusion Models برای اتصال نظریه نمونه‌برداری به مدل‌های مولد جدید وارد شده است.
