# محتویات بسته فصل ششم

- `image_processing_next_decade_ch06_fa.tex`: متن کامل LaTeX فصل
- `image_processing_next_decade_ch06_fa.pdf`: PDF نهایی ۴۶ صفحه‌ای
- `figures/`: شکل‌های PDF و PNG بازتولیدپذیر
- `scripts/generate_figures.py`: کد تولید شکل‌ها و آزمایش‌های عددی
- `outputs/`: خروجی‌های CSV کالیبراسیون و هندسه اپی‌پولار
- `requirements.txt`: وابستگی‌های پایتون
- `README_FA.md`: راهنمای کامپایل و اجرا

فایل با XeLaTeX و فونت‌های Amiri، DejaVu Serif و DejaVu Sans Mono آماده شده است. هیچ فایل فونتی در بسته قرار ندارد.
